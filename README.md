# Break Through Tech AI Program

## Overview
This repository contains projects and coursework completed during the **Break Through Tech AI Program**, a competitive 12-month AI program focused on machine learning, data science, and responsible AI.

I developed hands-on experience building machine learning models, exploring datasets, performing data preprocessing, and evaluating model performance using Python and industry-standard AI tools.

## Skills & Technologies
- **Languages:** Python
- **Libraries:** pandas, NumPy, scikit-learn, TensorFlow/Keras
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning:** 
  - Supervised Learning
  - Classification Models
  - Feature Engineering
  - Hyperparameter Tuning
  - Model Evaluation

## Projects

### Unit 5: Machine Learning Capstone: Census Income Prediction
**Goal:** Predict whether an individual's income exceeds $50K/year using demographic and employment data.

**Techniques Used:**
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Handling missing values
- One-hot encoding categorical features
- Feature engineering
- Model training and evaluation
- Hyperparameter optimization with GridSearchCV

**Models Implemented:**
- Decision Tree Classifier
- Logistic Regression
- Neural Network

**Evaluation Metrics:**
- Accuracy
- Precision
- Recall
- F1 Score
